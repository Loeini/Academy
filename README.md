## Docker

API simples com NestJS e PostgreSQL.

#### Comandos Principais

```bash
# Construir e iniciar containers
docker-compose up -d

# Ver status dos containers
docker-compose ps

# Ver logs da aplicação
docker-compose logs -f api
```
